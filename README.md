# friday

This are all my personal config files which I use for coding.

Dotfiles configuration is powered by [Dotbot](https://github.com/anishathalye/dotbot) ❤️

## Installation

1. Clone this repo

2. Run `./friday/install`

This will link all apps specified in `link` section of `install.conf.yaml` to this repo + install all dependencies from `Brewfile`.
